▷ A New Programming Paradigm ▷

MANIFOLD

A Primitive Shape as a Programming Language

I developed a new paradigm built from a single primitive: the manifold, a geometric shape whose curves and waveforms contain every program that can be derived from it. The gyroid rendered behind you is that primitive, live in your browser. Just as a piano contains every piano song ever written or yet to be written, a manifold contains every state, rule, and outcome of the system it encodes. Every entity in every game holds two numbers. A third coordinate is derived as z = x · y², and all three are folded into the natural period of the surface (2π) before evaluation. No arbitrary constants. No lookup tables. These five games are the proof of concept.

⚙ The Engine

Built from scratch. No commercial framework underneath. The five games on this portal don’t just share a server — they share something deeper. A common geometric foundation that makes the whole thing behave differently from anything built the conventional way.

The background is rendering live. The coordinates on the game cards are real. The golden ratio is in the spacing. The three surface equations are load-bearing. Pull on any thread and you find the same shape.

  • Pure WebGL2 — no engine dependency. GPU shaders written by hand.
  • Zero audio files — all sound is procedurally synthesized from the Web Audio API.
  • One coordinate system — every game event has a position in the same space.
  • Real-time leaderboards — scores, presence, and state live across all games simultaneously.
  • Browser-native — no installs. Works on any device with a GPU.
